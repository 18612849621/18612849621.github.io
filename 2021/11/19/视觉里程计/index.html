一、2D-2D：对极几何

1、对极约束

如图7-9，如果我们要求两帧图像I1I_1I1​, I2I_2I2​之间的运动，设第一帧到第二帧的运动为RRR，ttt。

  两个相机中心分别为O1O_1O1​，O2O_2O2​。现在，考虑I1I_1I1​中有一个特征点p1p_1p1​在I2I_2I2​中对应着特征点p2p_2p2​。如果匹配正确，就可以说明他们是同一个空间点在两个成像平面上的投影。

几何关系：O1p1⃗\vec{O_1p_1}O1​p1​​和O2p2⃗\vec{O_2p_2}O2​p2​​在三维空间中会相交于点PPP。由O1O_1O1​，O2O_2O2​，PPP三个点可以确定一个平面叫极平面（Epipolar plane）。O1O2O_1O_2O1​O2​连线与像平面I1I_1I1​，I2I_2I2​的交点分别为e1e_1e1​，e2e_2e2​。其中e1e_1e1​，e2e_2e2​称为极点（Epipoles），O1O2O_1O_2O1​O2​被称为基线。我们称极平面与两个像平面I1I_1I1​，I2I_2I2​之间的相交线l1l_1l1​，l2l_2l2​为极线（Epipolar line）。

几何分析：从第一帧的角度看，射线O1p1⃗\vec{O_1p_1}O1​p1​​是某个像素可能出现的空间位置（这个与之前单目针孔相机模型一样，没有深度信息所有射线上的点都会映射在此）则如果我们不知道PPP的位置，那么我们在第二幅图像上看到的e2p2⃗\vec{e_2p_2}e2​p2​​也是PPP可能出现的投影位置（因为他们在一个平面上，这条线投影出去一定包括点PPP，也就是射线O1p1⃗\vec{O_1p_1}O1​p1​​在第二个相机中的投影）然后我们通过特征点匹配确定了p2p_2p2​的像素位置，所以能够推断PPP的空间位置，以及相机的运动。

代数分析：

  在第一帧的坐标系下，设PPP的空间位置为

P=[X,Y,Z]TP = [X, Y, Z]^T P=[X,Y,Z]T

个人理解：这里的PPP我认为是按照第一帧的相机坐标系坐标

s1p1=KP,s2p2=K(RP+t)s_1p_1 = KP,\quad s_2p_2 = K(RP+t) s1​p1​=KP,s2​p2​=K(RP+t)

其中KKK为相机内参矩阵，RRR，ttt为两个坐标系的相机运动。

  这里使用了其次坐标表示像素点，在使用齐次坐标时，一个向量将等于它自身乘上任意的非零常数——用以上的方法表达一个投影关系。就像上式中s1p1s_1p_1s1​p1​和p1p_1p1​成投影关系，他们在齐次坐标的意义下的相等称作为尺度意义下相等(equal up to a scale)记作：

sp≃psp \simeq p sp≃p

那么上述的投影关系可以写作

p1≃KP,p2≃K(RP+t)p_1 \simeq KP, \quad p_2 \simeq K(RP+t) p1​≃KP,p2​≃K(RP+t)

几何意义：O1O_1O1​，PPP，O2O_2O2​三者共面，对极约束中同时包括了平移和旋转。其中有两个矩阵：1）基础矩阵FFF 2）本质矩阵EEE 于是进一步简化对极约束

E=t∧R,F=K−TEK−1,x2TEx1=p2TFp1=0E = t^\wedge R, \quad F=K^{-T}EK^{-1}, \quad x_2^TEx_1 = p_2^TFp_1 = 0 E=t∧R,F=K−TEK−1,x2T​Ex1​=p2T​Fp1​=0

对极约束简洁地给出了两个匹配点的空间位置关系。相机位姿估计问题变为以下两步：

1.根据匹配点的像素位置求出EEE或FFF

2.根据EEE或者FFF求出RRR，ttt。

实践中：EEE和FFF只相差了相机内参，而内参在SLAM中通常已知，所以我们往往按照EEE求解。

2、本质矩阵

本质矩阵E=t∧RE = t^\wedge RE=t∧R，是一个3x3的矩阵，内有9个未知数，满足以下条件：

（1）本质矩阵是由对极约束定义的。由于对极约束是等式为零的约束，所以对EEE乘以任意非零常数后，对极约束依然满足。我们把这件事情称为EEE在不同尺度下是等价的。

（2）根据E=t∧RE = t^{\wedge}RE=t∧R，可以证明，本质矩阵的EEE的奇异值必定是[σ,σ,0]T[\sigma,\sigma,0]^T[σ,σ,0]T的形式。这称为本质矩阵的内在性质。

（3）另外，由于平移和旋转各有3个自由度，故t∧Rt^{\wedge}Rt∧R共有6个自由度，但由于尺度等价性，故EEE实际上有5个自由度。（暂时理解为齐次矩阵矩阵那个添加的1那个尺度是一维度变量，但相当于没有影响的自由度）

3、八点法

4、单应矩阵

  如果场景中的特征点都落在同一个平面上（如墙、地面等），则可以通过单应性进行运动估计。

  单应矩阵通常描述处于共同平面上的一些点在两张图像之间的变换关系。设I1I_1I1​和I2I_2I2​有一堆匹配好的特征点p1p_1p1​和p2p_2p2​。这个特征点落在平面PPP上，设这个平面满足方程：

nTP+d=0n^TP + d = 0 nTP+d=0

整理一波

−nTPd=1-\frac{n^TP}{d} = 1 −dnTP​=1

然后联立这个式子

s1p1=KP,s2p2=K(RP+t)p2≃K(RP+t)≃K(RP+t⋅(−nTPd))≃K(R−tnTd)P≃K(R−tnTd)K−1p1s_1p_1 = KP,\quad s_2p_2 = K(RP+t)\\ p_2 \simeq K(RP+t)\\ \simeq K(RP+t\cdot(-\frac{n^TP}{d}))\\ \simeq K(R-\frac{tn^T}{d})P \\ \simeq K(R-\frac{tn^T}{d})K^{-1}p_1 s1​p1​=KP,s2​p2​=K(RP+t)p2​≃K(RP+t)≃K(RP+t⋅(−dnTP​))≃K(R−dtnT​)P≃K(R−dtnT​)K−1p1​

  于是我们得到了一个直接描述图像坐标p1p_1p1​和p2p_2p2​之间的变换，把中间这部分记为HHH，于是：

p2≃Hp1p_2 \simeq Hp_1 p2​≃Hp1​

  这样就可以由一对匹配点构造出两项约束（事实上是有三个约束，但是因为线性相关，只取前两个）于是自由度为8的单应矩阵可通过4对匹配特征点算出（在非退化的情况下，即这些特征点不能有三代点共线的情况），即求解一下的线性方程组（当h9=0h_9=0h9​=0时，右侧为零）：

  这种做法把HHH矩阵看成向量，通过解该向量的线性方程来恢复HHH，又称直接线性变换法（Direct Linear Transform，DLT）。与本质矩阵相似，求出单应矩阵以后需要对其进行分解，才可以得到相应的旋转矩阵RRR和平移向量ttt。

总结：单应性在SLAM中具有重要意义。当特征点共面或者相机发生纯旋转时，基础矩阵的自由度下降，出现退化。现实中的数据总要包含一些噪声，这时如果继续使用八点法求解基础矩阵，基础矩阵多余出来的自由度将会主要由噪声决定。为了能够避免退化现象造成的影响，同时估计基础矩阵FFF和单应矩阵HHH，找到最小的那个使用。
